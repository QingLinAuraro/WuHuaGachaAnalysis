# 模板图片说明

请将游戏截图裁剪后放入对应目录，用于页面识别（模板匹配）。

## 目录说明

### main/
主界面的**独特特征区域**截图（小图即可，约 100x100 ~ 300x200 px）
- 例如：右下角"招集"按钮图标
- 例如：界面上方的 Logo 或特殊装饰

### gacha_entrance/
主界面中**招集入口区域**的特征截图
- 例如：招集入口弹窗的标题栏

### gacha_home/
招集主页（选择卡池的界面）的特征截图
- 例如："活动招募"标签卡
- 例如："召集记录"按钮

### gacha_record/
召集记录列表页面的特征截图
- 例如：页面标题"召集记录"文字区域
- 例如：**"下一页"按钮**（命名为 `next_page.png`，用于翻页操作）
- 例如：单条记录条目示例

### gacha_detail/
召集记录详情页面的特征截图（如有）
- 例如：详情弹窗的特征区域

## 截图建议

- PNG 格式，清晰无水印
- 截取**特征明显且不随内容变化**的界面元素
- 每个目录放 1-3 张即可
- 推荐在 1280x720 分辨率下截图（与模拟器分辨率一致）
- 文件名随意，如 `title.png`、`button.png`、`tab.png`
